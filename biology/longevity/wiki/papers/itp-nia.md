# Interventions Testing Program (ITP)

NIA-sponsored peer-reviewed program that tests candidate
lifespan-extending compounds in a **genetically heterogeneous mouse
stock (UM-HET3)** in **parallel at three sites** (Jackson Lab, U. Michigan,
UT Health San Antonio) to guard against site/strain artifacts.

## Running since 2004
Public compound database + per-cohort survival data.

## Notable hits
- **Rapamycin** — 20–25% lifespan extension (both sexes, optimal doses).
  Inhibits mTOR / nutrient-sensing.
- **Acarbose** — sex-biased (larger effect in males). Acts as partial
  CR mimetic via carbohydrate-absorption delay, but mechanism appears
  more complex.
- **17α-estradiol** — male-only effect; non-estrogenic neuroprotective.
