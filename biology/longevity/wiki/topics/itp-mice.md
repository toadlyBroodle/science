# ITP and mouse lifespan data

The NIA Interventions Testing Program is the single most disciplined
in-vivo test bed for candidate lifespan-extending compounds.

## Core
- [[papers/itp-nia]] — program overview, UM-HET3, 3-site parallel tests
- [[papers/itp-mpd-portal]] — per-cohort data on Mouse Phenome Database

- [[papers/network-repurposing-aging]] — SHARP captures 90.9% of ITP

- [[papers/itp-epicatechin-halofuginone-mitoglitazone-2025]] — 2025 ITP
